# 🚀 Indian Beverages Industry Sales & Profitability Dashboard | Power BI

## 📌 Project Overview
Developed an interactive Sales & Profitability Dashboard to analyze over 1.5 lakh rows of data from the Indian Beverages Industry (FY 2020–2023).  

The objective of this project was to transform raw data into actionable insights and support business decision-making.

---

## 🎯 Business Problem
Businesses often struggle to track:
- Revenue and profit trends over time  
- Declining profit margins  
- Regional performance differences  

This dashboard solves these problems by providing a clear and interactive analytical view.

---

## 📊 Key Features
- 📈 Year-over-Year (YoY) Growth Analysis  
- 📅 Year-to-Date (YTD) Sales Tracking  
- 💰 Profit & Profit Margin % Analysis  
- 🌍 Region-wise Performance Insights  
- 📉 Identification of Profit Decline Trends  

---

## 🧠 Data Modeling
- Built a **Star Schema Data Model**  
- Created Fact and Dimension tables  
- Ensured efficient filtering and performance  

---

## ⚙️ DAX Measures Used
- YTD Sales  
- YoY Growth %  
- Profit Margin %  
- Total Revenue  
- Total Profit  

---

## 🎨 Dashboard Design
- Designed a **modern, neon-themed UI**  
- Focused on KPI visibility and user experience  
- Interactive filters and slicers for better analysis  

---

## 🔍 Key Insights
- Identified **declining profit margins despite rising sales**  
- Highlighted **high-cost regions impacting profitability**  
- Discovered **top-performing and underperforming segments**
